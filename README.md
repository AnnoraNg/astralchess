# Astral Chess Academy

Astral Chess Academy is my project for learning chess by creating a game that teaches me how to play.

Instead of only reading the rules, I can work through interactive lessons, practise each idea directly on the board, and then play against a bot to apply what I have learned.

## How it works

- 31 guided lessons organised into eight beginner-friendly sections
- Step-by-step coverage of the board, every piece, check, checkmate, castling, promotion, en passant, draws, openings, tactics, middlegames, and endgames
- Focused board exercises with legal-move validation and immediate feedback
- Short knowledge checks that explain why an answer is correct
- A searchable course map for revisiting individual topics
- Three bot difficulty levels for practising complete games
- Local progress, XP, lesson completion, ranks, and activity streaks
- A rulebook that separates official chess rules from practical strategy

## Play

[Open Astral Chess Academy](https://annorang.github.io/astralchess/)

## Run locally

```bash
npm install
npm run dev
```

## Test and build

```bash
npm test
npm run build
```

The project is built with React, Vite, and `chess.js`, and is deployed to GitHub Pages through GitHub Actions.
